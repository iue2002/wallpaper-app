# 小K壁纸

小K壁纸是一个桌面壁纸应用程序，支持多种壁纸来源和设置功能。

## 项目结构

- `Source code file-源码文件/` - 包含应用程序的源代码
- `win-x64-Release package-发布包/` - 包含应用程序的发布文件
- `小K壁纸.exe` - 主程序可执行文件

## 功能特性

- 支持多种壁纸来源
- 自动切换壁纸功能
- 适用于Windows10/11系统

## 使用说明

直接运行根目录下的`小K壁纸.exe`文件即可启动应用程序。或者从`win-x64-Release package-发布包/`目录中运行发布版本。
双击即可设置桌面背景，右键菜单有更多功能。
